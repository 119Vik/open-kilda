## Introduction

### How to Build Kilda Controller

From the base directory run these commands:

1. ```make build-base```
2. ```docker-compose build```

### How to run Kilda
 
From the base directory run these commands:

1. ```docker-compose up```

or

1. ```make up-test-mode```

### How to run floodlight-modules locally

From the base directory run these commands:

1. ```make build-floodlight```
2. ```make run-floodlight```
